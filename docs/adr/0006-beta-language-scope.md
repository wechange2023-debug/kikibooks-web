# ADR-0006 — 베타 콘텐츠 언어 스코프: 영어(English) 단일

**상태** Accepted
**날짜** 2026-05-14
**관련** `tasks/phase-04-book-dash-sync.json`, `tasks/phase-05-gdl-sync.json`(예정), `PLAN.md` 4절(베타 콘텐츠 스코프), `docs/adr/0005-book-dash-sync-strategy.md`

---

## 1. 배경

키키북스는 한국 유아(만 3~7세) 대상 **영어 그림책 e-라이브러리**다. 베타 출시 목표는 무료 합법 콘텐츠 1,300권 이상 적재이며, 핵심 학습 목표는 영어 노출이다.

원천 데이터에는 영어 외 언어가 다수 포함된다.
- Book Dash: 영어 원본 + 줄루(zu), 코사(xh), 아프리칸스(af), 소토(st), 츠바나(tn) 등 남아공 다국어 번역본
- GDL: 50+ 언어
- LibriVox: 영어 위주이나 다국어 포함

번역본까지 무차별 적재하면 카탈로그가 빠르게 1,300권을 넘기지만, 한국 학부모의 베타 사용자가 "왜 줄루어 책이 추천되지?"라는 혼란을 겪을 위험이 있다.

---

## 2. 결정

**베타 단계(Phase 0~1)에서 적재하는 모든 책은 `language = 'en'`이어야 한다.**

- 동기화 스크립트는 비영어 책을 INSERT 단계에서 명시적으로 skip하고 `skipped_by_language` 카운터에 집계한다.
- 단, `books.language` 컬럼 자체와 CHECK 제약은 다국어를 허용하는 현재 스키마를 유지한다. Phase 2 이후 확장 시 마이그레이션 없이 추가 가능하도록 한다.

---

## 3. 근거

### 3.1 제품 정체성
- 한국 부모의 핵심 페인포인트는 "영어 노출". 줄루·코사 등은 학습 동기와 무관하다.
- 추천 알고리즘(Phase 10 Screen 02 "오늘의 추천 5권") 신호가 언어 노이즈로 흐려진다.

### 3.2 큐레이션 비용
- 번역본까지 적재하면 카탈로그 권수가 명목상 5~10배 증가하지만, 영어 학습 가치가 0인 책이 다수 섞인다.
- 검색 결과 페이지(Screen 13 라이브러리)에서 필터를 매번 적용해야 하는 UX 부담이 발생한다.

### 3.3 라이선스 위험 없음
- CC BY 4.0은 번역본도 동일 라이선스로 배포된다. **언어 스코프를 좁히는 것은 라이선스 문제와 무관한 큐레이션 결정**이다.
- 본 ADR은 라이선스를 좁히는 것이 아니라 **적재 범위를 좁히는 것**이며, Hard Rule 1~5와 충돌하지 않는다.

### 3.4 Phase 04 권수 검증 기준 갱신
- 기존 `_index.json`은 phase-04에서 "200권 이상"을 요구하지만, 영어 only로 좁히면 ~50~60권 수준이다.
- 따라서 phase-04 검증 기준을 "50권 이상"으로 하향 조정하고, **Phase 04 + Phase 05(GDL) 합산으로 1,300권 목표를 달성**하는 방식으로 변경한다.
- `tasks/_index.json`의 phase-04 verification 항목 자체도 본 ADR과 동시에 갱신한다. (수정 시점 코멘트로 본 ADR 참조 명시)

### 3.5 미래의 확장 경로
- Phase 2 이후 협상 콘텐츠(JYBooks, 웅진주니어 등)는 한국어판이 포함될 가능성이 있으나, 이는 별도 ADR에서 다룬다.
- 다국어 확장 시점은 **베타 사용자 1,000명 + 추천 정확도 70% 이상 도달 후 검토** (KPI는 PLAN.md 13절 참조).

---

## 4. 결과

- `scripts/sync_book_dash.py`: language != 'en'은 skip + 카운트
- `scripts/sync_gdl.py` (Phase 05): GDL API 쿼리에 language=eng 필터 강제
- `tasks/_index.json` phase-04 verification 항목 갱신 (50권 이상)
- Phase 9(Screen 01 랜딩) 추천 로직에서 language 필터 불필요 — 모두 영어이므로

---

## 5. 재검토 트리거

- 베타 사용자가 "한국어 동화도 같이 볼 수 있으면 좋겠다"라고 반복적으로 피드백 (5건 이상)
- JYBooks 등 한국 출판사 협상 체결 → 한국어 콘텐츠 보유
- GDL이 영어 콘텐츠 1,000권 미달로 확인되어 다국어 보강이 필요해짐
